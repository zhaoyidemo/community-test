## 🏛️ AIP（改进提案）流程

### 什么是AIP？
AIP（AI for Good Improvement Proposal）是齐家AI的正式改进提案机制。

### 如何提交AIP？

**步骤1：想法讨论**
- 使用"💡 想法提交"模板创建Issue
- 在社区讨论1-2周

**步骤2：正式提案**  
- 使用"📜 AIP提案"模板创建Issue
- 填写完整的提案内容

**步骤3：社区评审**
- 社区讨论2-3周
- 补充细节和回答问题

**步骤4：理事会投票**
- 每月理事会会议审议
- 需要3/5（或4/5）票数通过

**步骤5：执行或归档**
- 通过 → 进入实施阶段
- 拒绝 → 说明理由并关闭

### AIP状态说明

- 🟡 `待审议` - 等待理事会审议
- 🟢 `已通过` - 理事会批准
- 🔴 `已拒绝` - 理事会否决
- 🔵 `实施中` - 正在执行
- 🟣 `已完成` - 执行完毕
